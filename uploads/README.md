# Armenian Plagiarism Detection System

## Master's Thesis - Information Security
### Gohar Tomeyan | Polytechnic Institute of Bragança

---

## 📋 Overview

This system is designed to detect plagiarism in Armenian texts using advanced Natural Language Processing (NLP) techniques, machine learning, and semantic analysis. It implements the methodologies described in the thesis "A text uniqueness checking system for Armenian Language".

### Key Features

- **Armenian Language Support**: Full support for Armenian alphabet, morphology, and grammar
- **Multi-level Plagiarism Detection**:
  - Exact text matching
  - Synonym-based detection
  - Semantic similarity analysis
  - Translation plagiarism detection
  - AI-generated content detection

- **Role-based Access**:
  - **Lecturers**: Full access with highlighted comparisons, admin panel for synonyms/themes
  - **Students**: Summary views, feedback system

- **Theme Detection**: Automatic identification of document themes (cryptography, cybersecurity, programming, etc.)

- **AI Content Detection**: ML-based detection of AI-generated text

- **Feedback System**: Communication between students and lecturers

- **Secure Authentication**: Password hashing with PBKDF2, SQL injection prevention
